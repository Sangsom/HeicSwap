1 · Product Overview

One-liner (subtitle): “PDF, Resize & Compress Photos”.

Problem: every iPhone shoots HEIC, so users keep hitting walls — a site rejects the photo, Android can’t open it, a form caps file size, or they need a PDF. Today’s apps upload to a server, bury it in ads, or spring a surprise weekly sub; the native Files/Shortcuts path is invisible and can’t do PDF/resize-to-size/RAW/WebP.

Vision: the default “just make it a normal file” app — fast, genuinely private (no server, ever), ad-free, honestly priced — and the on-device engine behind sibling utilities.

Audience: primary “Blocked Sender” (mainstream, non-technical); secondary Prosumer/RAW (served v1.1).

Anti-goals: no server/accounts · no ads · no third-party collection of your content · no editing · no dark patterns · no Android/web (v1).

3 · Feature Spec — MVP v1.0 (MoSCoW)

3.1 Photo & File Import M

Pick photos/files fast and start converting.

  • PhotosPicker multi-select (no full-library permission) + Files import; ordered queue.
  • Unsupported files flagged inline & skipped, never crash.
  • iCloud-optimized originals → “Downloading from iCloud…” via PHImageManager before convert.

3.2 On-Device Conversion Engine L

  • HEIC/HEIF/PNG/JPG → JPG/PNG/HEIC, 100% on-device (ImageIO), no network call.
  • Color profile preserved; no visible loss at “High”. CI test asserts zero network egress.
  • RAW input & WebP output are v1.1.

3.3 One-Tap Batch M

  • All queued items in one action with per-item progress (the “developing” reveal).
  • 100+ items stay responsive & memory-bounded; one failure doesn’t abort the batch (Retry).
  • Free up to a small batch; larger batches are Pro.

3.4 Image → Multi-Page PDF M Free ≤ batch

  • 2+ images → one multi-page PDF, in order, pages reorderable; single image → valid 1-page PDF.
  • Free within the free batch (≤5 images) — it’s our top acquisition term; Pro only for larger batches.

3.5 Resize / Compress to Target M target-size: Pro

  • Max dimension (px) or target file size (“under 2 MB”); quality iterated to hit target; presets “Web <2MB / Email / Original”.

3.6 Strip EXIF / Location S Pro

  • ON → no GPS/EXIF/maker-note in output (unit-tested); OFF default; persists.

3.7 Output & Sharing S

  • Save to Photos (add-only), Save to Files, or Share the batch; one tap from done.

3.8 Value-Gated Paywall (RevenueCat) L

  • Free: unlimited basic conversion up to N images/batch (launch N≈5), no watermark, no ads.
  • Pro gates: batch > N (conversion or PDF) · resize-to-size · strip-metadata (+ RAW/WebP v1.1). Small image→PDF is free.
  • Paywall: Annual $9.99 (default “Best value”) · Lifetime $19.99 · Weekly $1.99 (secondary, cancel anytime) · Restore.
  • Purchase unlocks instantly; entitlement cached for offline. Dismiss = stay free, no nag, no fake timer.

3.9 Onboarding S  ·  3.10 Settings S

  • ≤3 skippable screens (value → “never uploaded” → add photos). No account, no paywall in onboarding.
  • Settings: defaults, Manage/Restore subscription, literally-true privacy statement, Terms.
